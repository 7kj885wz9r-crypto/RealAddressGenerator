# Генератор Адресов

> **Языки**: [English](README.en.md) | [日本語](README.ja.md) | Русский | [中文](README.md)

Мощный инструмент для генерации адресов, поддерживающий множество типов генерации адресов, включая виртуальные адреса, тестовые адреса и т.д.

![Интерфейс Генератора Адресов](address_generator_en.png)

## 🌟 Онлайн Опыт

Посетите [https://addressgen.top](https://addressgen.top) для лучшего опыта!

## ✨ Возможности

- 🏠 **Генерация Реальных Адресов** - Генерация адресов в стандартных форматах различных стран
- 🌍 **Поддержка Многих Стран** - Поддержка США, Канады, Великобритании, Австралии, Германии, Франции
- 📱 **Адаптивный Дизайн** - Идеальная адаптация для настольных и мобильных устройств
- 🎯 **Пакетная Генерация** - Генерация нескольких адресов одновременно
- 📋 **Копирование Одним Кликом** - Быстрое копирование сгенерированных адресов
- 🔄 **Генерация в Реальном Времени** - Без необходимости обновления страницы
- 🎨 **Современный UI** - Чистый и красивый пользовательский интерфейс

## 🚀 Быстрый Старт

### Онлайн Использование
Посетите [https://addressgen.top](https://addressgen.top) напрямую для использования всех функций.

### Локальная Разработка

```bash
# Клонировать проект
git clone https://github.com/FunQz-com/address-generator.git
cd address-generator

# Установить зависимости
npm install

# Запустить сервер разработки
npm run dev

# Сборка для продакшена
npm run build
```

## 📖 Инструкции по Использованию

1. Выберите страну/регион для генерации адресов
2. Установите количество адресов для генерации
3. Нажмите кнопку "Генерировать Адреса"
4. Скопируйте или экспортируйте сгенерированные адреса

## 📝 Поддерживаемые Форматы Адресов

- 🇺🇸 Адреса США (включает штат, почтовый индекс, улицу)
- 🇨🇦 Канадские Адреса (включает провинцию, почтовый код)
- 🇬🇧 Адреса Великобритании (включает графство, почтовый индекс)
- 🇦🇺 Австралийские Адреса (включает штат, почтовый индекс)
- 🇩🇪 Немецкие Адреса (включает федеральную землю, почтовый индекс)
- 🇫🇷 Французские Адреса (включает регион, почтовый индекс)
