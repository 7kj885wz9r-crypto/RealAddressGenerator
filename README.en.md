# Address Generator

> **Languages**: English | [日本語](README.ja.md) | [Русский](README.ru.md) | [中文](README.md)

A powerful address generation tool that supports multiple types of address generation, including virtual addresses, test addresses, etc.

## ✨ Features

- 🏠 **Real Address Generation** - Generate addresses in standard formats for various countries
- 🌍 **Multi-Country Support** - Support for USA, Canada, UK, Australia, Germany, France
- 📱 **Responsive Design** - Perfect adaptation for desktop and mobile devices
- 🎯 **Batch Generation** - Generate multiple addresses at once
- 📋 **One-Click Copy** - Quickly copy generated addresses
- 🔄 **Real-time Generation** - No page refresh required
- 🎨 **Modern UI** - Clean and beautiful user interface

## 🚀 Quick Start

### Online Usage
Visit [https://addressgen.top](https://addressgen.top) directly to use all features.

### Local Development

```bash
# Clone the project
git clone https://github.com/FunQz-com/address-generator.git
cd address-generator

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📖 Usage Instructions

1. Select the country/region to generate addresses for
2. Set the number of addresses to generate
3. Click the "Generate Addresses" button
4. Copy or export the generated addresses

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox
- **Build Tool**: Vite
- **Deployment**: Static deployment support

## 📝 Supported Address Formats

- 🇺🇸 US Addresses (includes state, ZIP code, street)
- 🇨🇦 Canadian Addresses (includes province, postal code)
- 🇬🇧 UK Addresses (includes county, postcode)
- 🇦🇺 Australian Addresses (includes state, postcode)
- 🇩🇪 German Addresses (includes federal state, postal code)
- 🇫🇷 French Addresses (includes region, postal code)
